# DV：貓犬登記資料 — 資料說明（`petgov_raw.csv`）

> 資料路徑：`data_raw/petgov_raw.csv`  
> 角色：**依變數（Dependent Variable）**  
> 覆蓋範圍：台北市 12 行政區 × 2015–2024（民國104–113年），共 132 筆（含年成長率 2016 起）

---

## 欄位說明

| 欄位 | 型別 | 單位 | 說明 |
|------|------|------|------|
| `year` | int | 西元年 | 登記年份 |
| `district` | str | — | 行政區名稱（格式：`XX區`） |
| `dog_count` | int | 隻 | 當年底累計**登記狗隻數** |
| `cat_count` | int | 隻 | 當年底累計**登記貓隻數** |
| `cat_dog_ratio` | float | — | `cat_count / dog_count`（研究的核心 DV） |
| `cat_yoy` | float | % | 貓登記數年增率（vs. 上一年），2015 年為空值 |
| `dog_yoy` | float | % | 狗登記數年增率（vs. 上一年），2015 年為空值 |

---

## 資料來源

- **原始資料：** 台北市動物保護處／動物登記管理資訊系統
- **更新頻率：** 年度（每年底統計累計登記數）
- **處理腳本：** 無（原始即為區年格式）

---

## 資料品質注意事項

- 登記數為**累計存活登記**，不等於當年新登記；每年底數字包含歷年未報廢筆數。
- 部分低登記行政區（如南港、大同）在早年（2015–2016）`cat_count` 極少，比值可能不穩定。
- **研究期間請截取 2019–2024**（6 年），貓犬比值趨於穩定。
- `cat_yoy` / `dog_yoy` 僅供描述性統計，建模時通常使用 `cat_dog_ratio`。

## 使用範例（Python）

```python
import pandas as pd

dv = pd.read_csv("data_raw/petgov_raw.csv")
dv = dv[dv["year"].between(2019, 2024)]

# 確認 Panel 完整性（應有 12 區 × 6 年 = 72 筆）
assert len(dv) == 72, f"缺筆：{len(dv)}"
```
