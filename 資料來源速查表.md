## 📖 各檔案欄位說明

### DV：`petgov_panel_包含戶數.csv` — 貓犬登記數（含戶數）

來源：農委會寵物登記管理資訊網，12 區 × 11 年（2015–2025）。

| 欄位 | 型別 | 說明 |
|------|------|------|
| year | int | 西元年 |
| district | str | 行政區（如「中山區」） |
| dog_count | int | 當年度犬登記數 |
| cat_count | int | 當年度貓登記數 |
| cat_dog_ratio | float | 貓犬比（cat_count / dog_count） |
| cat_yoy | float | 貓登記數年增率（%），首年為空值 |
| dog_yoy | float | 犬登記數年增率（%），首年為空值 |
| house_count | int | 當年度戶數 |

### `age_district_panel.csv` — 年齡結構（幼兒＋老年）

來源：台北市民政局年齡結構 ODS（`data_raw/age/`），12 區 × 11 年（2015–2025）。

| 欄位 | 型別 | 說明 |
|------|------|------|
| ad_year | int | 西元年 |
| roc_year | int | 民國年 |
| district | str | 行政區 |
| total_pop | int | 該區總人口數 |
| pop_0_4 | int | 0–4 歲人口數 |
| ratio_0_4 | float | 幼兒比例（pop_0_4 / total_pop） |
| pop_65_plus | int | 65 歲以上人口數 |
| ratio_65_plus | float | 老年比例（pop_65_plus / total_pop） |

### `single_household_panel.csv` — 單人戶比例

來源：戶政 ODRP019（里級）彙總至區級，12 區 × 11 年（2015–2025）。

| 欄位 | 型別 | 說明 |
|------|------|------|
| year | int | 西元年 |
| district | str | 行政區 |
| single_total | int | 單獨生活戶數（區級加總） |
| ordinary_total | int | 共同生活戶數（區級加總） |
| ratio_single | float | 單人戶比例（single_total / (single_total + ordinary_total)） |

### `single_gender_panel.csv` — 單人戶性別分布

來源：戶政 ODRP019（里級）彙總至區級，12 區 × 10 年（2015–2024）。

| 欄位 | 型別 | 說明 |
|------|------|------|
| year | int | 西元年 |
| district | str | 行政區 |
| single_m | int | 男性單獨生活戶數 |
| single_f | int | 女性單獨生活戶數 |
| single_total | int | 單獨生活戶數合計 |
| diff_f_m | int | 女－男差值 |
| share_f | float | 女性佔單獨生活戶比例 |

### `edu_ratio_panel.csv` — 教育程度（高等教育比例）

來源：政府開放資料教育程度統計（`data_raw/education/`），12 區 × 11 年（2015–2025）。

| 欄位 | 型別 | 說明 |
|------|------|------|
| year | int | 西元年 |
| district | str | 行政區 |
| edu_higher_pop | int | 高等教育（大學以上）人口數 |
| edu_total_pop | int | 教育程度統計總人口數 |
| edu_ratio | float | 高等教育比例（edu_higher_pop / edu_total_pop） |

### `migration_panel.csv` — 遷入遷出

來源：台北市民政局遷入遷出 ODS（`data_raw/in_and_out/`），12 區 × 5 年（2020–2024）。

| 欄位 | 型別 | 說明 |
|------|------|------|
| district | str | 行政區 |
| year | int | 西元年 |
| move_in | int | 遷入人數 |
| move_out | int | 遷出人數 |
| birth | int | 出生人數 |
| death | int | 死亡人數 |
| net_migration | int | 淨遷移（move_in − move_out） |
| total_pop | int | 該區總人口 |
| IV_mig | float | 淨遷移率（net_migration / total_pop × 1000） |

### `rental_count_panel.csv` — 租賃件數

來源：內政部實價登錄租賃資料（`data_raw/rent/`），依區-年彙總，12 區 × 10 年（2015–2024）。

| 欄位 | 型別 | 說明 |
|------|------|------|
| year | int | 西元年 |
| district | str | 行政區 |
| rental_count | int | 該區該年租賃成交件數 |
| rental_ratio | float | 租賃件數佔全市比例 |

### `taipei_housing_type_panel.csv` — 房屋型態比例（wide format）

來源：內政部實價登錄買賣資料（`data_raw/building_type/`），依區-年彙總，12 區 × 12 年（2014–2025）。

| 欄位 | 型別 | 說明 |
|------|------|------|
| year | int | 西元年 |
| district | str | 行政區 |
| 公寓 | float | 公寓成交佔比 |
| 華廈 | float | 華廈成交佔比 |
| 住宅大樓 | float | 住宅大樓成交佔比 |
| 套房 | float | 套房成交佔比 |
| 透天厝 | float | 透天厝成交佔比 |
| 其他 | float | 其他類型成交佔比 |

### `taipei_housing_type_long.csv` — 房屋型態（long format）

與 `taipei_housing_type_panel.csv` 相同資料，轉為長格式，含筆數與佔比。

| 欄位 | 型別 | 說明 |
|------|------|------|
| year | int | 西元年 |
| district | str | 行政區 |
| building_type | str | 建物型態（公寓、華廈、住宅大樓、套房、透天厝、其他） |
| count | int | 該類型成交筆數 |
| proportion | float | 該類型佔該區該年成交比例 |

### `park_area_cross.csv` — 公園面積（截面）

來源：台北市公園資料，12 區截面資料。

| 欄位 | 型別 | 說明 |
|------|------|------|
| district | str | 行政區 |
| park_count | int | 公園數量 |
| park_area_m2 | float | 公園總面積（平方公尺） |
| park_area_km2 | float | 公園總面積（平方公里） |
| district_area_km2 | float | 行政區面積（平方公里） |
| park_area_pct | float | 公園面積佔行政區面積比例 |

### `vet_count_cross.csv` — 獸醫院數量（截面）

來源：Google Maps 搜尋彙總，12 區截面資料。

| 欄位 | 型別 | 說明 |
|------|------|------|
| district | str | 行政區 |
| vet_count | int | 獸醫院數量 |

### `taipei_female_business.csv` — 女性相關商家 POI（截面）

來源：Google Maps Places API 搜尋（美甲、美睫等關鍵字），12 區截面資料。

| 欄位 | 型別 | 說明 |
|------|------|------|
| district | str | 行政區 |
| keyword | str | 搜尋關鍵字（美甲、美睫等） |
| name | str | 商家名稱 |
| address | str | 商家地址 |

## 📋 資料對齊檢查清單

| 項目 | 規則 |
|------|------|
| 年份統一 | 民國年 + 1911 = 西元年 |
| 行政區名稱 | 統一為「XX區」，去掉「臺北市」前綴 |
| 時間點 | 人口/戶數取全年加總（里級彙總至區）；交易取全年彙總 |
| 研究期間 | 2015–2024（民國104–113），共10年 |
