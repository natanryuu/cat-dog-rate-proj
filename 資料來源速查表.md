## 🔧 需要下載的資料

### IV₁ 住宅成交坪數 + IV₄ 租賃件數（同一來源）

**來源：** 內政部實價登錄批次下載  
**網址：** https://plvr.land.moi.gov.tw/DownloadOpenData  

**Python 自動下載 URL 規則：**
```
https://plvr.land.moi.gov.tw/DownloadSeason?season={民國年}S{季}&type=zip&fileName=lvr_landcsv.zip
```

**範例：**
```
108S1 → https://plvr.land.moi.gov.tw/DownloadSeason?season=108S1&type=zip&fileName=lvr_landcsv.zip
108S2 → https://plvr.land.moi.gov.tw/DownloadSeason?season=108S2&type=zip&fileName=lvr_landcsv.zip
...
113S4 → https://plvr.land.moi.gov.tw/DownloadSeason?season=113S4&type=zip&fileName=lvr_landcsv.zip
```

**需下載範圍：** 108S1 ~ 113S4（最多24個ZIP）  
**ZIP 內容：**
- `a_lvr_land_a.csv` = **買賣**（用於 IV₁ 住宅坪數）
- `a_lvr_land_c.csv` = **租賃**（用於 IV₄ 租賃件數）

**處理方式：**
- IV₁：篩選住宅 → groupby(district, year) → mean(建物移轉總面積 × 0.3025)
- IV₄：groupby(district, year) → count()

### IV₃ 單人戶比例（分母：住宅總戶數）

**來源：** 內政部戶政司  
**網址：** https://www.ris.gov.tw/app/portal/346  

**操作步驟：**
1. 選「現住人口數、戶數」→「按區域別分」
2. 地區：臺北市 → 勾選12區
3. 時間：108–113年底（12月）
4. 下載Excel

**你已經有的資料：** 之前上傳的 `1_臺北市各區戶數_人口數按戶別分.csv`（108–114年）  
→ 直接用 `單獨生活戶 ÷ 合計` 算出單人戶比例  
→ `合計` 的戶數即為 IV₄ 的分母
