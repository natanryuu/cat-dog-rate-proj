## 📖 各檔案欄位說明

### DV：`petgov_raw.csv` — 貓犬登記數

來源：農委會寵物登記管理資訊網，12 區 × 11 年（2015–2025）。

| 欄位 | 型別 | 說明 |
|------|------|------|
| year | int | 西元年 |
| district | str | 行政區（如「中山區」） |
| dog_count | int | 當年度犬登記數 |
| cat_count | int | 當年度貓登記數 |
| cat_dog_ratio | float | 貓犬比（cat_count / dog_count） |
| cat_yoy | float | 貓登記數年增率（%），首年為空值 |
| dog_yoy | float | 犬登記數年增率（%），首年為空值 |

### IV₁：`iv1_housing_size_panel.csv` — 住宅平均坪數

來源：內政部實價登錄買賣資料（`data_raw/buy/`），依區-年彙總。

| 欄位 | 型別 | 說明 |
|------|------|------|
| year | int | 西元年（最早 1974，早期年份稀疏） |
| district | str | 行政區 |
| avg_housing_size | float | 該區該年買賣成交平均坪數 |
| median_housing_size | float | 該區該年買賣成交中位數坪數 |
| n_buy | int | 該區該年買賣成交筆數 |

### IV₃：`iv3_single_household_panel.csv` — 單人戶比例

來源：戶政 ODRP019（里級）彙總至區級，12 區 × 11 年（2015–2025）。

| 欄位 | 型別 | 說明 |
|------|------|------|
| year | int | 西元年 |
| district | str | 行政區 |
| single_total | int | 單獨生活戶數（區級加總） |
| ordinary_total | int | 共同生活戶數（區級加總） |
| ratio_single | float | 單人戶比例（single_total / (single_total + ordinary_total)） |

### IV₄：`iv4_rental_count_panel.csv` — 租賃件數

來源：內政部實價登錄租賃資料（`data_raw/rent/`），依區-年彙總，12 區 × 10 年（2015–2024）。

| 欄位 | 型別 | 說明 |
|------|------|------|
| year | int | 西元年 |
| district | str | 行政區 |
| rental_count | int | 該區該年租賃成交件數 |
| rental_ratio | float | 租賃件數佔全市比例 |

### 幼兒年齡結構：`age/age_district_panel.csv`

來源：台北市民政局年齡結構 ODS（`data_raw/age/`），由 `build_age_panel.py` 處理，12 區 × 11 年（2015–2025）。

| 欄位 | 型別 | 說明 |
|------|------|------|
| ad_year | int | 西元年 |
| roc_year | int | 民國年 |
| district | str | 行政區 |
| total_pop | int | 該區總人口數 |
| pop_0_4 | int | 0–4 歲人口數 |
| ratio_0_4 | float | 幼兒比例（pop_0_4 / total_pop） |

### 合併 Panel：`master_panel.csv`

將上述 DV 與所有 IV 依 year × district 合併，12 區 × 10 年（2015–2024）。

| 欄位 | 型別 | 說明 |
|------|------|------|
| year | int | 西元年 |
| district | str | 行政區 |
| cat_dog_ratio | float | 貓犬比（DV） |
| avg_housing_size | float | 住宅平均坪數（IV₁） |
| ratio_0_4 | float | 幼兒比例（幼兒年齡結構） |
| ratio_single | float | 單人戶比例（IV₃） |
| rental_ratio | float | 租賃件數佔全市比例（IV₄） |

### 原始資料：`odrp019_taipei_104_114.csv` — 戶政 ODRP019（里級）

來源：政府開放資料 ODRP019 API，由 `fetch_odrp019_taipei.py` 抓取，台北市里級戶數資料。

| 欄位 | 型別 | 說明 |
|------|------|------|
| statistic_yyy | int | 民國年 |
| site_id | str | 區域別（如「臺北市中山區」） |
| village | str | 村里名稱 |
| household_ordinary_total | int | 共同生活戶數 |
| household_business_total | int | 共同事業戶數 |
| household_single_total | int | 單獨生活戶數 |
| household_ordinary_m | int | 共同生活戶男性人口 |
| household_business_m | int | 共同事業戶男性人口 |
| household_single_m | int | 單獨生活戶男性人口 |
| household_ordinary_f | int | 共同生活戶女性人口 |
| household_business_f | int | 共同事業戶女性人口 |
| household_single_f | int | 單獨生活戶女性人口 |
| district_code | str | 區域代碼（目前為空） |
| year_ad | int | 西元年 |

### 原始資料：`single/opendata*Y010.csv` — 戶籍單人戶（全國里級）

來源：政府開放資料，中文欄位，涵蓋全國（非僅台北市），目前僅有 104–105 年。

| 欄位 | 型別 | 說明 |
|------|------|------|

| 區域別 | str | 縣市+區（如「新北市板橋區」） |
| 村里名稱 | str | 里名 |
| 共同生活戶_戶數 | int | 共同生活戶數 |
| 共同事業戶_戶數 | int | 共同事業戶數 |
| 單獨生活戶_戶數 | int | 單獨生活戶數 |
| 共同生活戶_男 | int | 共同生活戶男性人口 |
| 共同事業戶_男 | int | 共同事業戶男性人口 |
| 單獨生活戶_男 | int | 單獨生活戶男性人口 |
| 共同生活戶_女 | int | 共同生活戶女性人口 |
| 共同事業戶_女 | int | 共同事業戶女性人口 |
| 單獨生活戶_女 | int | 單獨生活戶女性人口 |

### 原始資料：`raw/{季度}/` — 實價登錄全國完整原始資料

來源：內政部不動產交易實價查詢服務網，每季一資料夾（`103S1`–`114S4`），共 48 季，每季約 197 個 CSV。涵蓋**全國 22 縣市**，包含買賣、預售屋、租賃三大類及其建物/土地/停車場明細。

**資料夾結構**：`data_raw/raw/{季度}/{縣市代碼}_lvr_land_{交易類型}.csv`，每季附帶 `manifest.csv`（檔案對照表）及 schema 檔。

**縣市代碼對照**：

| 代碼 | 縣市 | 代碼 | 縣市 | 代碼 | 縣市 | 代碼 | 縣市 |
|------|------|------|------|------|------|------|------|
| a | 臺北市 | b | 臺中市 | c | 基隆市 | d | 臺南市 |
| e | 高雄市 | f | 新北市 | g | 宜蘭縣 | h | 桃園市 |
| i | 嘉義市 | j | 新竹縣 | k | 苗栗縣 | m | 南投縣 |
| n | 彰化縣 | o | 新竹市 | p | 雲林縣 | q | 嘉義縣 |
| t | 屏東縣 | u | 花蓮縣 | v | 臺東縣 | w | 金門縣 |
| x | 澎湖縣 | z | 連江縣 | | | | |

**交易類型後綴**：

| 後綴 | Schema | 說明 |
|------|--------|------|
| `_a.csv` | schema-main.csv | 不動產買賣（成屋）主檔 |
| `_a_build.csv` | schema-build.csv | 買賣 — 建物明細 |
| `_a_land.csv` | schema-land.csv | 買賣 — 土地明細 |
| `_a_park.csv` | schema-park.csv | 買賣 — 停車場明細 |
| `_b.csv` | schema-main-sale.csv | 預售屋買賣主檔 |
| `_b_land.csv` | schema-land.csv | 預售屋 — 土地明細 |
| `_b_park.csv` | schema-park.csv | 預售屋 — 停車場明細 |
| `_c.csv` | schema-main-rent.csv | 不動產租賃主檔 |
| `_c_build.csv` | schema-build.csv | 租賃 — 建物明細 |
| `_c_land.csv` | schema-land.csv | 租賃 — 土地明細 |
| `_c_park.csv` | schema-park.csv | 租賃 — 停車場明細 |

**主檔主要欄位**（`_a.csv` 買賣為例，共 33 欄）：鄉鎮市區、交易標的、土地位置建物門牌、土地移轉總面積平方公尺、使用分區、交易年月日、交易筆棟數、移轉層次、總樓層數、建物型態、主要用途、主要建材、建築完成年月、建物移轉總面積平方公尺、建物現況格局（房/廳/衛/隔間）、有無管理組織、總價元、單價元平方公尺、車位類別、車位面積、車位總價、備註、編號、主建物面積、附屬建物面積、陽台面積、電梯、移轉編號。

> 注意：`data_raw/buy/` 和 `data_raw/rent/` 是從 `raw/` 中僅擷取台北市（代碼 `a`）的買賣與租賃主檔。`raw/` 為全國完整版。

### 原始資料：`buy/*.csv` — 實價登錄買賣成交

來源：內政部不動產交易實價查詢，每季一檔（`{季度}_a_lvr_land_a.csv`），共 40 檔（104S1–113S4）。

主要欄位：鄉鎮市區、交易標的、土地位置建物門牌、交易年月日、建物移轉總面積平方公尺、總價元、單價元平方公尺、建物型態、主要用途、建物現況格局（房/廳/衛）等共 33 欄。

### 原始資料：`rent/*.csv` — 實價登錄租賃

來源：內政部不動產交易實價查詢，每季一檔（`{季度}_a_lvr_land_c.csv`），共 40 檔（104S1–113S4）。

主要欄位：鄉鎮市區、交易標的、土地位置建物門牌、租賃年月日、建物總面積平方公尺、總額元、單價元平方公尺、建物型態、主要用途、有無附傢俱等共 29 欄。

## 📋 資料對齊檢查清單

| 項目 | 規則 |
|------|------|
| 年份統一 | 民國年 + 1911 = 西元年 |
| 行政區名稱 | 統一為「XX區」，去掉「臺北市」前綴 |

## 🗂️ 資料夾結構

```
cat-dog-rate-proj/
├── download_all_data.py           # 實價登錄下載 + IV₁/IV₄ 處理
├── build_age_panel.py             # 年齡結構 ODS → Panel 處理
├── fetch_odrp019_taipei.py        # 戶政 ODRP019 抓取（IV₃ 原始）
├── 資料來源速查表.md

    ├── petgov_raw.csv                  # DV：貓犬登記（含 YoY）
    ├── master_panel.csv                # 合併 Panel（DV + 全部 IV）
    ├── iv1_housing_size_panel.csv      # IV₁：住宅坪數 Panel（區年級）
    ├── iv3_single_household_panel.csv  # IV₃：單人戶比例 Panel（區年級）
    ├── iv4_rental_count_panel.csv      # IV₄：租賃件數 Panel（區年級）
    ├── odrp019_taipei_104_114.csv      # IV₃ 原始：單人戶（里級，104–114年）

    │       ├── {縣市}_lvr_land_{類型}.csv
    │       ├── manifest.csv            #   檔案對照表

    ├── buy/                            # 實價登錄買賣原始 CSV（40個，僅台北市）
    ├── rent/                           # 實價登錄租賃原始 CSV（40個，僅台北市）
    ├── age/                            # 年齡結構 ODS（12個）+ 處理後 Panel
    │   └── age_district_panel.csv
    ├── single/                         # 戶籍單人戶原始 CSV（2個）
    └── zip_cache/                      # 實價登錄 ZIP 快取
```
