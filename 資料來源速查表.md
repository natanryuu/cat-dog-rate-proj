## 📋 資料對齊檢查清單

| 項目 | 規則 |
|------|------|
| 年份統一 | 民國年 + 1911 = 西元年 |
| 行政區名稱 | 統一為「XX區」，去掉「臺北市」前綴 |
| 時間點 | 人口/戶數取全年加總（里級彙總至區）；交易取全年彙總 |
| 研究期間 | 2015–2024（民國104–113），共10年 |
